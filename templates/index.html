Tutorial: Organizador de Archivos

Aprende a usar la aplicación para organizar tus archivos automáticamente

1. Descarga e inicia la aplicación

Descarga el archivo ejecutable desde la raíz del repositorio, en la ruta: dist/Organizador.exe. Puedes acceder al repositorio en GitHub desde el siguiente enlace: https://github.com/YonyChaparro/Organizador-de-archivos.

Luego, haz doble clic en Organizador.exe para abrir la aplicación.

2. Selecciona la carpeta a organizar

Una vez abierta la app, haz clic en "Seleccionar Carpeta". Se abrirá una ventana para escoger la carpeta que deseas organizar.

Imagen 2: Ventana para seleccionar carpeta

3. Personaliza las categorías

Antes de iniciar el proceso, puedes editar las extensiones que pertenecerán a cada categoría de archivos (como Texto, Imagen, Audio, etc.). Simplemente modifica las listas de extensiones visibles en la interfaz.

Imagen 3: Edición de categorías y tipos de archivo

4. Organiza tus archivos

Una vez todo esté listo, haz clic en "Organizar Archivos". La app creará automáticamente carpetas por tipo de archivo y moverá los archivos correspondientes.

Imagen 4: Botón para organizar archivos y resultado final

5. Resultado final

En la carpeta seleccionada encontrarás una nueva carpeta llamada "Archivos organizados" con subcarpetas como "Texto", "Audio", "Video", etc. Tus archivos estarán allí distribuidos según su tipo.

Imagen 5: Estructura generada tras la organización
